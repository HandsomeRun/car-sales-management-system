系统介绍

这是一个基于零框架理念开发的汽车销售管理信息系统，采用 Node.js 原生 HTTP 模块和 TypeScript 构建。 系统支持汽车信息管理、销售订单管理等核心功能，并实现了10,000+ 并发用户的设计目标。

技术特点
  • ✅ 零框架实现 - 使用 Node.js 原生 HTTP 模块，不依赖 Express/Fastify
  • ✅ TypeScript 开发 - 类型安全，提升代码质量
  • ✅ 滑动时间窗口限流 - 支持 1000 req/s/IP，防止恶意攻击
  • ✅ 内存数组模拟 - 简化实现，聚焦项目管理流程展示
  • ✅ 分层架构设计 - Router → Service → DAO 清晰分离
  • ✅ RESTful API - 标准化的 API 设计
  • ✅ 响应式前端 - 原生 HTML + CSS + ES6，支持移动端
